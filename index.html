Mes projets

  • SpotAPI 1
    • HTML
    • CSS
    • JavaScript
    • group : 2
    • schedule : 2W
    • school

    Ce projet avait pour but d'apprendre à utiliser une API. Pour cela, nous avons utilisé l'API de Spotify pour récupérer des informations sur des artistes, des albums et des chansons. De plus nous avons développé un système de favoris pour sauvegarder les recherches de l'utilisateur dans le local storage. Enfin, nous avons utilisé un endpoint de l'API pour permettre à l'utilisateur de contrôler sa musique sur son compte Spotify depuis notre site.
